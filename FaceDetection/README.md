# Face Detection

Face Detection 모델을 만든 내용을 정리하였습니다.

Tensorflow의 <a href="https://github.com/tensorflow/models/tree/master/research/object_detection">Object Detection API</a>의 pretrained model(ssd mobilenet-v1)을 사용하였습니다.



## Environments

- Linux Ubuntu 18.04
- Tensorflow 1.14
- Python 3.7.3
